# The Importance of Belonging

This repository contains a short educational website about the **importance of belonging**. It explores how a sense of belonging impacts human identity, security, and survival, using historical and literary examples.

## Overview

The website has **5 sections**:

1. **Title / Introduction** – Explains the main idea: belonging gives purpose and identity.
2. **Reason 1** – Dehumanization in literature (*The Metamorphosis*) and Holocaust camps.
3. **Reason 2** – Legal and physical persecution: Nuremberg Laws, Kristallnacht, and Rwanda Genocide.
4. **Reason 3** – Social isolation: boycotts, Judenrein policies, and international inaction in crises.
5. **Conclusion** – Highlights why belonging is fundamental and how isolation increases vulnerability.
